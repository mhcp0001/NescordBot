# PRを作成

- Read @.github/PULL_REQUEST_TEMPLATE.md に従うこと
- push時は`git push -u origin <branch_name>` のように `--set-upstream` を指定すること
- コマンドの例: `gh pr create --draft --title "title" --body "body"`
